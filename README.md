# 深圳国际美术馆七展手机导览 2.0

打开 `index.html` 即可预览（线上版：https://glimmer2077.github.io/masters-realm-gallery/ ）。

这一版包含：

- 七个展览的统一卡片墙：《大师之境》《国宝归来》《缣素绵延》《东方根脉与当代转化》《徐冰：字在乐园》《新链：艺术的未来》《闪烁的1/∞》。
- 每个卡片可点击打开详情弹窗。
- 《大师之境》保留 71 件作品墙和 12 件重点作品（图片全部本地化）。
- 其他六个展览为重点文物/艺术家/概念/体验导览卡（图标卡，无需图片）。
- 新增 2.5D 游览地图，可点击展区跳转。
- `index.embedded.html` 为单文件嵌入版，已把全部 71 张本地作品图转为 base64。

使用方式：
1. 保持 `index.html` 和 `assets/` 文件夹在同一层级。
2. 打开 `index.html`。

《大师之境》卡片直接读取本地图片；其余六展为图标导览卡。

当前已本地缓存作品图：71 / 71（全部齐全）。

每件作品的图都在 `assets/artworks/<作品id>.jpg`（少数为 `.webp`），并在 `artworks-data.json` 中设置了 `localImage`。若你之后有更清晰的现场照片或官方图，替换同名文件即可。

`index.embedded.html` 是单文件版，已经把全部 71 张作品图转成 base64 嵌入，可完全离线、单文件打开。

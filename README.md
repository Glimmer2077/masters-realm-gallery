# 大师之境｜手机博物馆导览（修正版）

这版修复了上一版下载后部分图片仍然缺失的问题：`assets/artworks/` 已经随 ZIP 一起打包，并且 `index.html` 会优先读取本地图片。

使用方式：
1. 解压整个 ZIP。
2. 保持 `index.html` 和 `assets/` 文件夹在同一层级。
3. 打开 `index.html`。

图片加载顺序：本地图片 → 原始远程图片/来源页预览 → 占位卡。这样即使某个馆方预览页不能直连，也不会出现大块空白。

当前已本地缓存作品图：71 / 71（全部齐全）。

每件作品的图都在 `assets/artworks/<作品id>.jpg`（少数为 `.webp`），并在 `artworks-data.json` 中设置了 `localImage`。若你之后有更清晰的现场照片或官方图，替换同名文件即可。

`index.embedded.html` 是单文件版，已经把全部 71 张作品图转成 base64 嵌入，可完全离线、单文件打开。
